
---------------------------
How To Use 0-add_integer.py
---------------------------

This module defines an integer combining function named ``add_integer(a, b=98)``.

-----
Usage
-----

```add_integer(...)``` Returns the addition of two arguments,
for numbers it's the equivalance of using the ``+`` operator.

::

	>>> add_integer = __import__('0-add_integer').add_integer
	>>> add_integer(2, 3)
	5

::

	>>> add_integer(2, -3)
	-1

::

	>>> add_integer(2.9, 0.2)
	2

::

	>>> add_integer(2)
	100

::
